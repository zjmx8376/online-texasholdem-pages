# online-texasholdem-pages

本仓库用于归档与发布多个独立的 HTML 页面，内容涵盖德州扑克相关的工具、演示或参考页面。这些页面均为自包含的静态文件，不依赖特定后端服务，可直接在浏览器中打开使用。

## 目录说明

```
.
├── README.md          # 本说明文件
├── pages/             # 存放所有 HTML 页面文件
│   ├── example.html   # 示例页面
│   └── ...            # 其他归档页面
└── assets/            # 公共资源（CSS、JS、图片等，如有）
```

- `pages/` 目录下每个 HTML 文件为一个独立归档页面，文件名应反映其功能或用途。
- `assets/` 目录用于存放多个页面共享的资源文件，避免重复。

## 页面归档说明

本仓库仅收录以下类型的页面：

- 德州扑克游戏规则说明、概率计算、牌型参考等工具页面
- 学习或教学用途的演示页面
- 与德州扑克相关的独立小工具或界面原型

所有页面均以静态 HTML 形式存放，不包含任何动态后端逻辑。页面内可能引用外部资源（如 CDN 上的库），但不会与特定域名或服务绑定。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并在本 README 中适当位置添加说明。
- 更新页面：直接替换对应文件，保持文件名不变。
- 删除页面：移除文件并更新 README 中的相关条目。
- 请确保每个页面在提交前已通过基本功能测试，避免损坏的链接或脚本错误。

## 使用方式

1. 克隆本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
